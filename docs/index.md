# Agentic Engineering Guidebook

A field guide to the era of human-agent software development.

Version 1.0 | 2026

## What Is Agentic Engineering?

Agentic engineering is the practice of integrating agents as primary producers of work artifacts in software development, with humans steering intent, architecture, and judgment.

In this model:

- Humans set direction, define quality, and make high-stakes decisions.
- Agents execute scoped tasks, generate artifacts, and iterate quickly.
- Quality improves through explicit harnesses, review, and feedback loops.

The shift is not "AI helps me code faster." It is about designing workflows so human and agent strengths are both used deliberately.
